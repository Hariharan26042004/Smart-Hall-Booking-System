# Smart Hall Booking System

An Android-based hall booking application developed using Java.

## Overview

This application allows users to check hall availability and send booking requests through a mobile interface.  
The admin can review and approve booking requests efficiently.

## Features

- User Login
- View Available Halls
- Send Booking Requests
- Admin Approval System
- Booking Status Management
- Simple Android User Interface

## Technologies Used

- Java
- Android Studio
- Firebase / MySQL
- XML

## Project Type

Android Application

## Future Enhancements

- User Registration
- Online Payment Integration
- Push Notifications
- Real-time Availability Updates
